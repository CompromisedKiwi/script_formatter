# 🐮小牛马 - 剧本格式化工具
# 介绍
- 这是一个用于格式化剧本文档的工具
- 支持的文件格式：`.docx, .doc`

# feature
- **标红并加粗**对话（对话段落中冒号后的内容，中英冒号均可）
   - 对话中括号内的内容保持默认
- 首字符为 **@** 时，**替换**首字符为 **△**
- 设置**阿拉伯数字**开头、**“人物”** 开头、**“第\*集”** 的段落为**标题3**样式
   - 其他段落保持原段间距
- 统一设置字体为 **宋体**，字号为 **小四**
- 可一次性格式化多个文件

# 使用方法
1. 启动：双击`小牛马-剧本格式化工具.exe`
2. 在弹出的文件选择对话框中，选择要格式化的剧本文档（可多选）
3. 程序将在同一目录下的`已格式化`文件夹中生成新文件
4. 格式化完成后，会弹出提示框显示保存路径
